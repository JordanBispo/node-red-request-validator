## Requirements 
* node-red  version >=1.0.0

## Instalation
#### Install via Node-RED Manage Palette
`node-red-request-validator`

#### Install via npm
```bash
$ cd ~/.node-red
$ npm install node-red-request-validator
# then restart node-red
```
