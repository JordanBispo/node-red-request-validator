## Requirements 
* node-red  version >=1.0.0

## Installation
#### Install via Node-RED Manage Palette
`node-red-contrib-request-validator`

#### Install via npm
```bash
$ cd ~/.node-red
$ npm install node-red-contrib-request-validator
# then restart node-red
```
